課程回顧與反思

🎨

3D 物件製作

使用 monster mash 及 polycam 製作 3D 物件，將平面想像轉化為空間存在。

🤖

AI 遊戲生成

使用 google ai studio 生成網頁遊戲，體驗提示詞工程與程式邏輯的結合。

🚀

GitHub 部署

使用 github 上傳並發布網頁，將作品推向全世界的網路空間。

📂

首頁整合

使用 Gemini 生成最終的學習成果首頁，完成系統性的整合。

總結心得

其中，製作 3D 物件很有趣，Google AI Studio 生成網頁遊戲令我印象深刻。

我覺得最難的是網頁上傳到 GitHub 的過程，細節非常瑣碎，碰到了非常多的難關。但透過不斷的嘗試與解決問題，最後得到滿滿的成就感，這才是學習中最珍貴的部分。
